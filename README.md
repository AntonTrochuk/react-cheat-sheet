# Шпаргалка по REACT

## 1. АНАТОМИЯ ПРОЕКТА: РАЗБИРАЕМ ПО ПОЛОЧКАМ

- my-app/
  - 📁 node_modules/ — 🧳 БАГАЖ
    Ты его не трогаешь, тут лежат все библиотеки

  - 📁 public/ — 🖼 ВИТРИНА
    Файлы, которые уходят в браузер "как есть"
    - 📄 index.html — 🏠 ГЛАВНЫЙ ФАЙЛ
      Единственная HTML-страница

    - 📄 favicon.ico — 🔖 Иконка

  - 📁 src/ — 💡 МОЗГ
    Здесь ты будешь работать 99% времени
    - 📁 components/ — 🧩 КУБИКИ
      Твои переиспользуемые кнопки, шапки, карточки

    - 📁 pages/ — 📄 СТРАНИЦЫ
      То, что видит пользователь (Главная, О нас)

    - 📄 App.jsx — 🧠 ЦЕНТР УПРАВЛЕНИЯ
      Корневой компонент

    - 📄 index.js — 🚪 ВХОДНАЯ ДВЕРЬ
      Точка входа в приложение

    - 📄 index.css — 💅 ОБЩИЙ СТИЛЬ
      Шрифты, фон, базовые настройки

    - 📄 package.json — 📜 ОПИСЬ
      Какие библиотеки стоят и как запускать проект

    - 📄 README.md — 📘 ИНСТРУКЦИЯ
      Памятка для разработчиков
