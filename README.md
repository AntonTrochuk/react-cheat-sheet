# Шпаргалка по REACT

## 1. АНАТОМИЯ ПРОЕКТА: РАЗБИРАЕМ ПО ПОЛОЧКАМ

my-app/
┣ 📁 node_modules/ 🧳 БАГАЖ — ты его не трогаешь, тут лежат все библиотеки
┣ 📁 public/ 🖼 ВИТРИНА — файлы, которые уходят в браузер "как есть"
┃ ┣ 📄 index.html 🏠 ГЛАВНЫЙ ФАЙЛ — единственная HTML-страница
┃ ┗ 📄 favicon.ico 🔖 Иконка
┣ 📁 src/ 💡 МОЗГ — здесь ты будешь работать 99% времени
┃ ┣ 📁 components/ 🧩 КУБИКИ — твои переиспользуемые кнопки, шапки, карточки
┃ ┣ 📁 pages/ 📄 СТРАНИЦЫ — то, что видит пользователь (Главная, О нас)
┃ ┣ 📄 App.jsx 🧠 ЦЕНТР УПРАВЛЕНИЯ — корневой компонент
┃ ┣ 📄 index.js 🚪 ВХОДНАЯ ДВЕРЬ — точка входа в приложение
┃ ┗ 📄 index.css 💅 ОБЩИЙ СТИЛЬ — шрифты, фон, базовые настройки
┣ 📄 package.json 📜 ОПИСЬ — какие библиотеки стоят и как запускать проект
┗ 📄 README.md 📘 ИНСТРУКЦИЯ — памятка для разработчиков
